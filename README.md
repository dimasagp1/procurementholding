# Procurement System (procurementversiA)

This is the Procurement and Purchase Request System (Version 3) integrated with budget tracking and validation.

## About the Project
This application is built using the Laravel framework. It manages Departmental Purchase Requests (PR) with master item settings, unit of measure (UOM) configurations, and departmental budget enforcement.

## System Features
- **Purchase Requests (PR):** Manage, approve, export, and track departmental purchase requests.
- **Budget Tracking:** Track monthly departmental and global budgets.
- **Master Data Settings:** Manage master items, UOM, departments, and users.
- **Role-based Access Control:** Set permissions and dashboard access based on roles.
